Agent

.. py:class:: Agent(host[, port = 80][, https = False])

   :param host: The fully qualified domain name, hostname or an IPv4 address of the Agent
   :param port: Set a custom port number (0 - 65535), defaults to ``80``
   :param https: Set to ``True`` to require HTTPS, defaults to ``False``
   :type host: str
   :type port: int
   :type https: bool
